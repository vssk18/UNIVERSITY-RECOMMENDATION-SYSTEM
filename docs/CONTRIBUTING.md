# Contributing

Contributions are welcome! This project is meant to be **practical and realistic**, not just a toy script.

## Ways to Contribute

- Improve the **dataset**:
  - Add universities
  - Fix ranking or IELTS data
  - Add new course types or regions
- Tune the **scoring logic**:
  - Adjust weights for GPA, GRE, IELTS, etc.
  - Refine tier boundaries
  - Improve bucket rules
- Improve the **CLI UX**:
  - Better prompts
  - Additional modes
  - Export options (e.g., CSV / JSON)
- Add **tests**:
  - Unit tests for scoring and bucket logic
  - Sanity checks for dataset
